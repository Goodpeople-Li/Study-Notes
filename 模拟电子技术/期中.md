# 模拟电子技术期中复习

## 第一章 半导体元件

1.  基本符号或下标分别采用大写或者小写的含义
   - 大写字母、大写下标表示直流量
   - 大写字母、小写下标表示交流有效值
   - 大写字母上面加点、小写下标表示正弦向量
   - 小写字母、大写下标表示总瞬时值
   - 小写字母、小写下标表示交流分量瞬时值

1. 本征半导体：纯净的半导体材料，没有掺杂任何杂质
2. 杂质半导体：在本征半导体中掺入少量杂质原子形成的半导体材料
3. 扩散运动：由于载流子浓度分布不均匀，载流子从高浓度区向低浓度区移动的现象称为扩散运动
4. 漂移运动：载流子在内电场作用下定向移动的现象称为漂移运动
5. 势垒电容$C_b$：是由于空间电荷区的宽度的变化引起的，与结电阻并联，由于节电阻反向偏置的时候极大，所以此时的势垒电容不容忽视，高频时有较大影响，综上所示，==势垒电容再反向偏置的时候影响较大==
6. 扩散电容$C_d$: 由于载流子在结区的扩散引起的电容，与PN结的体电阻并联，==在正向偏置时影响较大==
    注：==高频应用的时候需要考虑PN结的结电容问题，PN结的总电容$C_j = C_b + C_d$, 与PN结的体电阻相并联。

1. 死区电压（开启电压）
2. $U_{on}$: 正向导通压降。
3. $I_S$: 反向饱和电流。
4. $U_{BR}$: 反向击穿电压
5. 二极管方程：$i_D = I_S \left( e^{\frac{U_D}{nU_T}} - 1 \right)$
   注：$i_D$为二极管电流，$U_D$为二极管两端电压，$U_T$为温度的电压当量，常温下$U_Y \approx  26mV$。
1. 最大整流电流$I_{F}$：在周期内允许通过二极管的最大正向平均电流。
2. 最高反向工作电压$U_{R}$：在周期内允许加在二极管上的最大反向电压。
3. 反向电流$I_{R}$：在反向工作时通过二极管的电流。
4. 最高工作频率$f_{max}$：二极管在规定的工作条件下能正常工作的最高频率。
5. 稳压管：==工作在反向击穿区的二极管==，利用其稳压特性实现稳压功能的器件称为稳压管。
6. 三极管：发射结正偏，集电结反偏。
7. $\beta$（电流放大系数）：$\beta = \frac{I_C}{I_B}$
   - $\overline\beta$：共射直流电流放大系数: $\overline\beta = \frac{I_{C(Q)}}{I_{B(Q)}}$， $\overline{\beta} = \left(\frac{\overline{\alpha}}{1- \alpha}\right)$
   - $\alpha$: 共基直流电流放大系数: $\alpha = \frac{\Delta i_C}{\Delta i_E}$
   - $\overline\alpha$: 共基电流放大系数: $\overline\alpha = \frac{I_C}{I_E}$
1. 三极管工作在截止区时，发射结和集电结均反偏。$u_{BE} < 0$,$u_{BC} < 0$
2.  三极管工作在放大区时，发射结正偏，集电结反偏。$u_{BE} > 0$, $u_{BC} < 0$
3.  三极管工作在饱和区时，发射结和集电结均正偏。$u_{BE} > 0$, $u_{BC} > 0$
4.  反向饱和电流$I_{CBO}$:发射极开路，cb之间的反向电流
5.  穿透电流$I_{CEO}$: 基极开路时ce之间的电流
   $I_{CEO} = (1 + \overline\beta)I_{CBO}$

6.  结型效应管：d漏极、g栅极、s源极
7.  场效应管的输出特性：可变电阻区、恒流区、截止区
8.  夹断电压$U_{CS(off)}$; 饱和漏极电流$I_{DSS}$
9.  结型场效应管的转移特性曲线可以近似用以下公式表示：
    $i_D = I_{DSS} \left(1 - \frac{U_{GS}}{U_{GS(off)}}\right)^2$
10. 绝缘栅型场效应管的转移特性曲线可以近似用以下公式表示：
    $i_D = I_{DO}\left(\frac{u_{GS}}{U_{GS(th)}} - 1\right)$
   其中，$I_{DO}$为当$U_{GS} = 2U_{GS(th)}$时的漏极电流。
11. 低频跨导$g_m$：$g_m = \frac{\Delta i_D}{\Delta u_{GS}}$


## 放大电路的基本原理和分析方法
1. 通频带BW：高频和低频段分别下降至中频段放大倍数的$\frac{1}{\sqrt{2}}$时所对应的频率范围，称为通频带，记为BW。
2. $r_{re}$的近似估算公式：$r_{re} \approx \frac{26mV}{I_{EQ} (mA)} + r_{bb'}$ 
   注：$r_{bb'}$默认为300Ω。
3. 放大电路的输出电阻的求法：将放大电路的输入信号源置零（短路电压源，开路电流源），再看输出端的等效电阻，即为放大电路的输出电阻。
4. 共基接法比共射接法的输入电阻小，是后者的$\frac{1}{1 + \beta}$。输出电阻大，是后者的$1 + \beta$,适合用于高频放大电路。
5. 三种基本组态的总结
   - 共射电路的输入电阻和输出电阻比较适中，一般用这个就行。
   - 共集电路的输入电阻较大，输出电阻较小，适合用作多级放大电路的输入极、输出极以及隔离用的中间级。也就是电压跟随器。
   - 共基电路的输入电阻较小，输出电阻较大，适合用于宽频放大器，也能用作恒流源。
