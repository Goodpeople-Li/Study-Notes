# 一.基本语法
1. **标题**
   # 一级标题
   ## 二级标题
   ### 三级标题  以此类推……（最多支持六级标题）

2. **引用**
   >类似于注释可以实现对应效果
    >>这个是二级的引用，可以实现嵌套

3. **列表**
   1. 无序列表
    - 列表一
    + 列表二
    * 列表三
   2. 有序列表
   3. 嵌套
   4. TodoList
   - [x] a
   - [x] b
   - [x] c
  
4. **表格**
   | 左对齐 | 居中对齐 | 右对齐 |
   | :----- | :------: | -----: |
   | a      |    b     |      c |

5. **段落**
   - 换行 ―― 两个空格以上之后回车
   - 分割线 ―― 三个*
       ***

   - 字体
  
    |     字体      |    代码    |
    | :-----------: | :--------: |
    |    *斜体*     |    * *     |
    |   ==高亮==    |   == ==    |
    |   **粗体**    |   ** **    |
    | ***粗斜体***  |  *** ***   |
    |   ~~删除~~    |   ~~ ~~    |
    | <u>下划线</u> | `<u> </u>` |
    - 脚注
