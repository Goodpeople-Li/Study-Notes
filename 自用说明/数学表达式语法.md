# 数学表达式语法速查（Markdown + KaTeX）

本页速查适用于常见 Markdown 渲染器（VS Code 预览、Markdown All in One、Markdown Preview Enhanced 等）采用的 KaTeX/LaTeX 数学语法。

## 基本用法
- 内联公式：用一对 `$...$` 包裹。
	- 示例：`$a^2+b^2=c^2$` → $a^2+b^2=c^2$
- 块级公式：独占一行，用一对 `$$...$$`。
	- 示例：

		```
		$$
		a^2+b^2=c^2
		$$
		```

## 常用语法
- 上下标：`x_i^2`、`x_{ij}^{n+1}` → $x_i^2,\ x_{ij}^{n+1}$

关于箭头（→）的输入方法：
- 在数学环境中建议用 LaTeX：`\to` 或 `\rightarrow`（示例：`x\to y` → $x\to y$）。
- 直接插入符号：复制粘贴 “→”。
- 编辑器/输入法：有的会把 `->` 自动替换为 →，或通过插件实现。
- 操作系统字符面板：Windows 可用 Win + . 打开符号面板；macOS 用 Control + Command + Space；Linux 使用字符映射工具。
- 网页/HTML：使用实体 `&rarr;` 或 `&#8594;`。
- 分式：`\frac{a}{b}` → $\frac{a}{b}$

注：`\frac` 是 "fraction" 的缩写。
- 字母上面的线：`\overline`
- 根号：`\sqrt{x}`、`\sqrt[n]{x}` → $\sqrt{x},\ \sqrt[n]{x}$
- 关系/集合：`\le \ge \neq \approx \in \notin \subset \supset \cup \cap` → $\le\ \ge\ \neq\ \approx\ \in\ \notin\ \subset\ \supset\ \cup\ \cap$
- 运算符：
	- 求和：`\sum_{i=1}^n i` → $\sum_{i=1}^n i$
	- 连乘：`\prod_{k=1}^n a_k` → $\prod_{k=1}^n a_k$
	- 积分：`\int_{0}^{1} f(x)\,dx` → $\int_{0}^{1} f(x)\,dx$
	- 极限：`\lim_{x\to 0} f(x)` → $\lim_{x\to 0} f(x)$
- 向量/装饰：`\vec{v}`、`\bar{x}`、`\hat{\theta}` → $\vec{v},\ \bar{x},\ \hat{\theta}$
- 文本夹杂：`\text{最大值}` → $\text{最大值}$（在数学环境内插入普通文字）

## 矩阵、分段与对齐
- 矩阵（方括号 bmatrix；常见还有 pmatrix 圆括号、vmatrix 单竖线）：

	代码：
	```
	$$
	\begin{bmatrix}
	a & b \\
	c & d
	\end{bmatrix}
	$$
	```

	效果：
	$$
	\begin{bmatrix}
	a & b \\
	c & d
	\end{bmatrix}
	$$

- 分段函数：

	代码：
	```
	$$
	f(x)=
	\begin{cases}
		x^2, & x\ge 0\\
		-x,  & x<0
	\end{cases}
	$$
	```

	效果：
	$$
	f(x)=
	\begin{cases}
		x^2, & x\ge 0\\
		-x,  & x<0
	\end{cases}
	$$

    case表示分段函数的情况。

- 多行对齐（`&` 指定对齐位置，`\\` 换行）：

	代码：
	```
	$$
	\begin{aligned}
	a &= b + c \\
		&= d + e
	\end{aligned}
	$$
	```

	效果：
	$$
	\begin{aligned}
	a &= b + c \\
		&= d + e
	\end{aligned}
	$$


    aligned表示一个数学对齐环境（multi-line alignment），用于在数学模式内把多行公式按 `&` 指定的位置对齐。

   - 公式书写提示（LaTeX 括号）：`\left` 与 `\right` 可让括号随内容（如分式、矩阵）自适应高度。
       - 圆括号自适应：`$ \left( \frac{a}{b} \right) $` → $ \left( \frac{a}{b} \right) $
       - 仅一侧需要分隔：`$ \left. \dfrac{df}{dx} \right|_{x=0} $` → $ \left. \dfrac{df}{dx} \right|_{x=0} $
       - 条件竖线：`$ \left\{ x \middle| x>0 \right\} $` → $ \left\{ x \middle| x>0 \right\} $
       - 简单内联可直接用普通括号：`( e^{UD/(nUT)} - 1 )`
       - 注意：在 Markdown 中实际写单反斜杠即可（示例中为展示效果使用了转义）。

## 排版与细节
- 空格：`\,`（小） `\;`（中） `\quad` `\qquad`（大）。常见如积分里用 `\,dx`。
- 编号：KaTeX 支持 `\tag{1}` 手工编号，如 `$$ E=mc^2 \\ \tag{1} $$`。
- 字体：`\mathbf{A}` 粗体，`\mathbb{R}` 黑板粗体，`\mathcal{L}` 花体。

## 常见坑
- `$` 转义：写字面美元符号用 `\$`。
- 下划线：普通文本里的 `_` 若被误解析，可用反引号包裹代码片段，或在数学环境内用上下标语法。
- 块级 `$$...$$` 应独占一行，避免与同一行文字混排。
- 长公式建议使用 `aligned` 分行；GitHub 等部分渲染器对自动编号有限，必要时用 `\tag{}`。

## 希腊字母与变化量符号速查
- 小写希腊字母：`\alpha \beta \gamma \delta \epsilon \zeta \eta \theta \iota \kappa \lambda \mu \nu \xi \pi \rho \sigma \tau \upsilon \phi \chi \psi \omega` → $\alpha\ \beta\ \gamma\ \delta\ \epsilon\ \zeta\ \eta\ \theta\ \iota\ \kappa\ \lambda\ \mu\ \nu\ \xi\ \pi\ \rho\ \sigma\ \tau\ \upsilon\ \phi\ \chi\ \psi\ \omega$
- 大写希腊字母：`\Gamma \Delta \Theta \Lambda \Xi \Pi \Sigma \Upsilon \Phi \Psi \Omega` → $\Gamma\ \Delta\ \Theta\ \Lambda\ \Xi\ \Pi\ \Sigma\ \Upsilon\ \Phi\ \Psi\ \Omega$
- 变化量符号：
  - 小写 delta（δ，微小变化）：`\delta x` → $\delta x$
  - 大写 Delta（Δ，总变化）：`\Delta x` → $\Delta x$
  - 微分 d：`dx` → $dx$
  - 偏微分 ?：`\partial x` → $\partial x$
  - 梯度 ?：`\nabla f` → $\nabla f$


## 量纲
- 欧姆（ohm）：`\Omega` → $\Omega$
